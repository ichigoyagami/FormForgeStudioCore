# FormForge Studio roadmap

## Phase 1 — reproducible source foundation

- Pin a supported Blender upstream revision.
- Apply FormForge changes as reviewable source patches.
- Bundle the interface, theme, icon and splash at build time.
- Produce a Windows installer and portable build.
- Add smoke tests and diagnostic export.

## Phase 2 — stable modelling workflow

- Native interactive Extrude, Bevel, Bridge and Inset controls.
- Persistent operator parameters and viewport manipulators.
- Maya-style component modes and marking menus.
- Native Multi-Cut with snapping, loop insertion and preview.
- Selection rings and loops for vertices, edges and faces.

## Phase 3 — retopology

- Persistent stored Live Surface.
- Separate solid retopology mesh.
- Crash-safe cached surface acceleration.
- Native Quad Draw point, face, strip, relax and tweak operations.
- Edge-loop insertion, deletion, symmetry and undo transactions.
- Viewport navigation during every retopology operation.

## Phase 4 — complete FormForge experience

- Source-native startup workspace and panel layout.
- Channel Box, Attribute Editor, Outliner and Modeling Toolkit.
- Sculpting, UV, rigging and animation workflows.
- FormForge project format evolution with `.blend` migration support.
- Installer, updater, release signing and regression suite.
