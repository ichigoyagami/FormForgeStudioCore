# FormForge Studio Core

FormForge Studio is a professional 3D content-creation application developed as a GNU GPL source fork of Blender, with an independently implemented Maya-inspired workflow.

This is the new canonical FormForge source project. The automated Windows build checks out a pinned Blender upstream revision, applies the reviewed FormForge source patch, installs the built-in FormForge interface and branding, compiles the executable, and creates installer and portable artifacts.

## Current integration

- FormForge executable branding
- Built-in FormForge UI enabled at startup
- Maya-inspired graphite interface and navigation
- Modelling menus, shelf, Channel Box and Modeling Toolkit
- Live Surface and experimental Quad Draw foundation
- Interactive modelling-operation wrappers
- Center Pivot and D/V transform controls
- FormForge splash and Windows icon
- Windows installer and portable package
- `.forge` project association foundation
- Developer log and diagnostics

## Development direction

The next phase moves Multi-Cut, Live Surface, Quad Draw, modelling manipulators and persistent operation options into tested native source modules. Unstable experimental tools must not be presented as production-ready.
